# <a name="azure-storage-client-tools"></a>Ferramentas de cliente do armazenamento Azure

Utilizadores de armazenamento do Windows Azure frequentemente pretende que possam ver/interagir com os seus dados através de uma ferramenta de cliente de armazenamento do Windows Azure. Nas tabelas abaixo, podemos lista um número de ferramentas que permitem-lhe para o fazer. Vamos colocar um "X" em cada bloco se fornece a capacidade de quer enumerar e/ou aceder à produção de dados. A tabela também mostra se as ferramentas de é gratuito, ou não. "Avaliação" indica que não existe uma versão de avaliação gratuita, mas o produto completo não é gratuito. "Y/N" indica que está disponível uma versão gratuitamente, enquanto uma versão diferente está disponível para compra.

Fornecemos apenas um instantâneo das ferramentas de cliente de armazenamento do Windows Azure disponíveis. Estas ferramentas podem continuar a evoluir e crescem em sequência de funcionalidade. Se existirem correções ou atualizações, deixe num comentário para diga-nos. O mesmo se aplica que se souber qual das ferramentas de devessem ser aqui - podemos seria satisfeitos adicioná-los.
