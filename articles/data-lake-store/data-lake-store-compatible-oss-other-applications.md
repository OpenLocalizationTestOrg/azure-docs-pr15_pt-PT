# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-store"></a>Abrir as aplicações de origem de dados grande que funcionam com o arquivo de Lake de dados do Azure

Para as aplicações na tabela abaixo, são suportadas apenas as versões disponíveis com a distribuição listada. Para obter informações sobre que versões destas aplicações estão disponíveis com HDInsight, consulte o artigo [o controlo de versões do HDInsight componente](../hdinsight/hdinsight-component-versioning.md).
