# <a name="azure-active-directory-b2c-file-support-requests"></a>Azure Active Directory B2C: Pedidos de suporte de ficheiro

Pode arquivar os pedidos de suporte para B2C Azure Active Directory (Azure AD) no portal do Azure através dos seguintes passos:

1. [Siga estes passos para navegar para o pá de funcionalidades B2C no portal do Azure](active-directory-b2c-app-registration.md#navigate-to-the-b2c-features-blade).
2. Mudar do seu inquilino B2C para outro inquilino que tenha uma subscrição do Azure associada. Normalmente, o último é o inquilino do funcionário ou o inquilino predefinido criado por si quando se inscreveu para uma subscrição do Azure. Para saber mais, consulte o artigo [como uma subscrição do Azure está relacionada com Azure AD](active-directory-how-subscriptions-associated-directory.md#how-an-azure-subscription-is-related-to-azure-ad).

    ![Suporte - inquilinos do parâmetro](./media/active-directory-b2c-support/support-switch-dir.png)

3. Depois de mudar de inquilinos, clique em **Ajuda + suporte**.

    ![Suporte - ajuda + suporte](./media/active-directory-b2c-support/support-support.png)

4. Clique em **novo pedido de suporte**.

    ![Suporte - novo](./media/active-directory-b2c-support/support-new.png)

5. No pá **Noções básicas** , utilize estes detalhes e clique em **seguinte**.

    - **Tipo de problema** é **técnico**.
    - Selecione a **subscrição**adequado.
    - **O serviço** está **Do Active Directory**.
    - Escolha o **plano de suporte**de adequado. Se não tiver uma, pode inscrever-se para uma [aqui](https://azure.microsoft.com/en-us/support/plans/).

    ![Suporte - Noções básicas](./media/active-directory-b2c-support/support-basics.png)

6. No pá **problema** , utilize estes detalhes e clique em **seguinte**.

    - Escolha o nível de **gravidade** adequado.
    - **Tipo de problema** é **B2C**.
    - Selecione a **categoria**adequadas.
    - Descreva o problema no campo de **Detalhes** . Fornece detalhes, como o nome do inquilino de B2C, a descrição do problema, mensagens de erro, correlação IDs (se disponível) e assim sucessivamente.
    - No campo **período de tempo** , forneça a data e hora (incluindo fuso horário) em que ocorreu o problema.
    - Em **envio de ficheiros**, carregar todas as capturas de ecrã e ficheiros que pensa que quiser dar assistência a resolver o problema.

    ![Suporte - problema](./media/active-directory-b2c-support/support-problem.png)

7. Na pá **informações de contacto** , adicione as informações de contacto. Clique em **Criar**.

    ![Suporte - contacto](./media/active-directory-b2c-support/support-contact.png)

8. Depois de submeter o pedido de suporte, pode monitorizá-la ao clicar em **Ajuda + suporte** Startboard e, em seguida, **os pedidos de suporte de gerir**.

## <a name="known-issue-filing-a-support-request-in-the-context-of-a-b2c-tenant"></a>Problema conhecido: arquivar um pedido de suporte no contexto de um inquilino B2C

Se não atendidas passo 2 descritos acima e tente criar um pedido de suporte no contexto do seu inquilino B2C, verá o seguinte erro.

> [AZURE.IMPORTANT]
> Não tentativa para se inscrever para uma nova subscrição Azure no inquilino do B2C.  

![Suporte - nenhuma subscrição](./media/active-directory-b2c-support/support-no-sub.png)
