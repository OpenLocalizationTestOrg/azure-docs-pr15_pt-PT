# <a name="enabling-azure-active-directory-identity-protection"></a>Activar a proteção de identidade do Azure Active Directory 

Azure Active Directory identidade Protection é uma nova capacidade que fornece uma vista consolidada em atividades de início de sessão suspeitas e potenciais vulnerabilidades e com as notificações, recomendações remediação e políticas de risco ajuda-o a proteger a sua empresa. 

O serviço de Deteta suspeitas atividades de utilizador final e identidades privilegiados (admin) com base num sinais como ataques de força bruta, fuga credenciais, inicie sessão ins a partir de localizações não está familiarizadas, infectados dispositivos, para proteger contra estas actividades em tempo real. Ainda mais importante, com base nestas actividades suspeitas, é calculada uma gravidade do risco de utilizador e políticas de risco podem ser configuradas e proteger automaticamente as identidades da sua organização. Para obter mais detalhes, consulte o artigo [Protecção de identidade do Azure Active Directory](active-directory-identityprotection.md).


Este tópicos mostra como ativar o Azure Active Directory identidade proteção.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Passos para ativar o Azure Active Directory identidade Protection 


1. [Início de sessão](https://ms.portal.azure.com/) para o seu portal Azure como administrador global. 

1. No portal do Azure, clique em **Marketplace**.

    ![Criar] (./media/active-directory-identityprotection-enable/01.png "Criar")

1. Na lista de aplicações, clique em **segurança + identidade**.

    ![Criar] (./media/active-directory-identityprotection-enable/02.png "Criar")

1. Clique em **proteção de identidade do Azure AD**.

    ![Criar] (./media/active-directory-identityprotection-enable/03.png "Criar")

1. No pá **Azure AD identidade proteção** , clique em **Criar**.

    ![Criar] (./media/active-directory-identityprotection-enable/04.png "Criar")
