# <a name="manage-passwords-in-azure-active-directory"></a>Gerir palavras-passe no Azure Active Directory

Se for um administrador, pode repor a palavra-passe de um utilizador no Azure Active Directory (Azure AD) no portal do Azure clássico. Clique no nome do seu diretório e na página utilizadores, clique no nome do utilizador e na parte inferior do portal clique em **Repor palavra-passe**.

O resto deste tópico abrange o conjunto completo das capacidades da gestão de palavra-passe Azure AD suporta, incluindo:

- Alterar **palavra-passe personalizada** permite aos utilizadores finais ou os administradores alterar a palavra-passe expirada ou não expiraram sem chamar um administrador ou suporte técnico para obter suporte.
- Reposição **personalizada de palavra-** permite aos utilizadores finais ou administradores para repor a palavra-passe automaticamente sem chamar um administrador ou suporte técnico para obter suporte. Reposição personalizada de palavra-requer o Azure AD Premium ou Basic. Para mais informações, consulte o artigo [edições do Azure Active Directory](active-directory-editions.md).
- **Reposição de palavra-passe iniciado pelo administrador** permite que o administrador para repor a um utilizador final ou outro administrador palavra-passe de no interior do Azure portal clássico.
- Informações de administradores de atribua de **relatórios de atividade de gestão de palavra-passe** para a atividade de reposição e registo de palavra-passe que ocorram na sua organização.
- **Palavra-passe escrita não consolidada** permite a gestão das palavras-passe de no local a partir da nuvem para que todos os cenários acima podem ser executados ao ou em nome da, federado ou palavra-passe de utilizadores sincronizados. Palavra-passe escrita não consolidada requer Azure AD Premium. Para mais informações, consulte o artigo [Introdução ao Azure Active Directory Premium](active-directory-get-started-premium.md).

> [AZURE.NOTE]
> Azure AD Premium está disponível para clientes chinês utilizando a instância de nível mundial do Azure AD. Azure AD Premium atualmente não é suportada no serviço Microsoft Azure operado pela 21Vianet na China. Para mais informações, contacte-nos durante o [Azure Active Directory fórum](https://feedback.azure.com/forums/169401-azure-active-directory/).
